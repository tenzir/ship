# pi-tenzir-ship

Pi skill package for [tenzir-ship](https://github.com/tenzir/ship) changelog and
release workflows.

## Install

```sh
pi install npm:pi-tenzir-ship
```

## Use

Inside pi:

```text
/skill:tenzir-ship
```
